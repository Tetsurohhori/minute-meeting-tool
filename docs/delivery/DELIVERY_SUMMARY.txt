🚀 クイックスタート（5ステップ）
================================================================================

1. 仮想環境の作成とパッケージインストール
   $ python3 -m venv venv
   $ source venv/bin/activate
   $ pip install -r requirements.txt

2. 環境変数の設定
   $ cp .env.example .env
   $ nano .env  # OpenAI APIキー等を設定

3. 認証情報の配置（Google Drive使用時）
   $ cp /path/to/your/credentials.json ./credentials.json

4. 初回データ取り込み
   $ python scripts/update_vector_store.py

5. アプリケーション起動
   $ streamlit run app/main.py
   → http://localhost:8501 にアクセス
